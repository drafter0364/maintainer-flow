# Security Policy

Maintainer Flow is a maintainer-assistance tool. It reads repository metadata, diffs, issue text, release notes, and optional model configuration. Treat all analyzed repository content as untrusted input.

## Reporting a vulnerability

Please do not open a public issue for a vulnerability. Email the maintainers or use GitHub private vulnerability reporting when it is available for the repository.

Include:

- Affected version or commit.
- Clear reproduction steps.
- Impact and affected workflow.
- Whether secrets, repository write permissions, or published artifacts are involved.

## Supported versions

This repository currently supports the latest released minor version only.

## Maintainer response target

We aim to acknowledge valid private reports within 7 days and provide a remediation plan or status update within 30 days.
